## Overview

HazeOver Advanced is a focus enhancement application that dims all background windows when you are working in the foreground application, reducing visual distractions and helping you concentrate on the task at hand. It creates a subtle spotlight effect where the active window stands out clearly against a dimmed backdrop, providing visual focus that reduces the cognitive load of a cluttered multi-window desktop. The dimming is automatic, adjusting as you switch between applications without any manual activation.

Automatic dimming applies a configurable darkness overlay to all windows except the currently focused application, creating a visual hierarchy that naturally draws attention to the active window. The dimming intensity is adjustable from barely perceptible to near-blackout, enabling users to find the level that enhances focus without making background content inaccessible when needed. The transition animation is smooth and subtle.

Multi-display awareness handles dimming independently per display, enabling focused work on one screen while maintaining full visibility on reference displays. Display-specific dimming settings accommodate workflows where different monitors serve different purposes — primary work on one display, reference material visible on another.

App-specific settings override the global dimming for applications that should never be dimmed or should always be visible regardless of focus. This handles scenarios where monitoring applications, communication tools, or reference windows need to remain fully visible while other background content is dimmed.

For anyone who works with multiple windows open simultaneously and finds the visual clutter distracting, HazeOver Advanced is the focus tool that creates visual calm by dimming everything except what matters right now. Its automatic, subtle operation enhances concentration without requiring conscious activation or workflow changes.
